核心实现
Core Implementation


Matrix adjugateInverse(const Matrix& A) {
    const int n = A.rows;
    
    // Calculate determinant
    double det = determinant(A);
    if (abs(det) < 1e-10) {
        throw "Matrix is singular";
    }
    
    // Calculate cofactor matrix
    Matrix cofactor(n, n);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            cofactor.data[i][j] = cofactor(A, i, j);
        }
    }
    
    // Transpose to get adjugate
    Matrix adjugate = transpose(cofactor);
    
    // Scale by 1/det(A)
    Matrix inverse(n, n);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            inverse.data[i][j] = adjugate.data[i][j] / det;
        }
    }
    
    return inverse;
}

Algorithm Steps

1. Determinant Calculation: Compute det(A) and check if matrix is invertible
2. 余子式矩阵：计算每个元素的余子式：(-1)^(i+j) × M_ij
2. Cofactor Matrix: Calculate cofactor for each element: (-1)^(i+j) × M_ij
3. 伴随矩阵：转置余子式矩阵得到 adj(A)
3. Adjugate Matrix: Transpose the cofactor matrix to get adj(A)
4. 行列式缩放：将伴随矩阵乘以 1/det(A) 得到逆矩阵
4. Scale by Determinant: Multiply adjugate by 1/det(A) to get inverse
5. 验证：检查 A × A⁻¹ = I 在容差范围内
5. Verification: Check that A × A⁻¹ = I within tolerance

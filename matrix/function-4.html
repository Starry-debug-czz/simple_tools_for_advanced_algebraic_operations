核心实现
Core Implementation


Matrix gaussJordanInverse(const Matrix& A) {
    const int n = A.rows;
    
    // Create augmented matrix [A|I]
    Matrix aug(n, 2*n);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            aug.data[i][j] = A.data[i][j];
        }
        aug.data[i][n+i] = 1; // Identity matrix
    }
    
    // Gauss-Jordan elimination
    for (int i = 0; i < n; i++) {
        // Pivot selection
        int pivot = i;
        for (int j = i+1; j < n; j++) {
            if (abs(aug.data[j][i]) > abs(aug.data[pivot][i])) {
                pivot = j;
            }
        }
        
        // Check for singularity
        if (abs(aug.data[pivot][i]) < 1e-10) {
            throw "Matrix is singular";
        }
        
        // Row swap if needed
        if (pivot != i) {
            swapRows(aug, i, pivot);
        }
        
        // Normalize pivot row
        double pivotVal = aug.data[i][i];
        for (int j = i; j < 2*n; j++) {
            aug.data[i][j] /= pivotVal;
        }
        
        // Eliminate other rows
        for (int j = 0; j < n; j++) {
            if (j != i) {
                double factor = aug.data[j][i];
                for (int k = i; k < 2*n; k++) {
                    aug.data[j][k] -= factor * aug.data[i][k];
                }
            }
        }
    }
    
    // Extract inverse from right half
    Matrix inverse(n, n);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            inverse.data[i][j] = aug.data[i][n+j];
        }
    }
    
    return inverse;
}

算法步骤
Algorithm Steps

1. 增广矩阵：将单位矩阵附加到 A 形成 [A|I]
1. Augmented Matrix: Create [A|I] by appending identity matrix to A
2. 主元选择：在当前列中寻找具有最大绝对值的行
2. Pivot Selection: Find row with maximum absolute value in current column
3. 行操作：归一化主元行并消去其他行
3. Row Operations: Normalize pivot row and eliminate other rows
4. 提取逆矩阵：增广矩阵的右半部分成为 A⁻¹
4. Extract Inverse: Right half of augmented matrix becomes A⁻¹
5. 验证：检查 A × A⁻¹ = I 在容差范围内
5. Verification: Check that A × A⁻¹ = I within tolerance

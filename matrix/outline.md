## Page Breakdown

### 1. index.html - Homepage
**Purpose**: Welcome users and introduce the platform
**Sections**:
- Navigation bar with language toggle
- Hero section with 3D holographic Earth
- Floating particle background
- "Enter the Matrix" call-to-action button
- Brief introduction to linear algebra tools

**Key Features**:
- Animated cosmic background
- Typewriter effect for main title
- Liquid metal button with electromagnetic pulse
- Particle vortex scroll indicator

### 2. directory.html - Function Directory
**Purpose**: Display all available linear algebra functions
**Sections**:
- Navigation bar
- Function grid (3x4 layout)
- Category color coding
- Hover popups with code previews
- Return to homepage option

**Key Features**:
- Interactive function icons
- Holographic popup windows
- Category-based color system
- Smooth transitions between functions

### 3. function-X.html - Individual Function Pages
**Purpose**: Execute specific linear algebra operations
**Common Structure**:
- Navigation bar with breadcrumbs
- Split layout (code left, input right)
- Interactive matrix input
- Real-time validation
- Results display with options

**Key Features**:
- Syntax-highlighted code display
- Matrix input with dimension validation
- Error handling with neon alerts
- Step-by-step process visualization
- Export and navigation options

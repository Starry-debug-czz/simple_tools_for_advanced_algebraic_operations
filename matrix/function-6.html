核心实现
Core Implementation


int matrixRank(const Matrix& mat) {
    const int rows = mat.rows, cols = mat.cols;
    Matrix temp = mat; // Work on a copy
    int rank = 0;
    
    for (int col = 0; col < cols && rank < rows; col++) {
        // Find pivot row
        int pivot = rank;
        for (int i = rank + 1; i < rows; i++) {
            if (abs(temp.data[i][col]) > abs(temp.data[pivot][col])) {
                pivot = i;
            }
        }
        
        // Skip if pivot is too small
        if (abs(temp.data[pivot][col]) < 1e-10) {
            continue;
        }
        
        // Swap rows if needed
        if (pivot != rank) {
            swapRows(temp, rank, pivot);
        }
        
        // Eliminate other rows
        for (int i = rank + 1; i < rows; i++) {
            double factor = temp.data[i][col] / temp.data[rank][col];
            for (int j = col; j < cols; j++) {
                temp.data[i][j] -= factor * temp.data[rank][j];
            }
        }
        
        rank++;
    }
    
    return rank;
}

算法步骤
Algorithm Steps

1. 初始化：从秩 = 0 开始，在矩阵副本上操作
1. Initialize: Start with rank = 0 and work on matrix copy
2. 列处理：对每列，寻找具有最大绝对值的主元
2. Column Processing: For each column, find pivot with maximum absolute value
3. 行消元：使用行操作在主元下方创建零
3. Row Elimination: Use row operations to create zeros below pivot
4. 秩递增：每个成功的主元使秩增加 1
4. Rank Increment: Each successful pivot increases rank by 1
5. 结果：秩等于阶梯形式中非零行的数量
5. Result: Rank equals number of non-zero rows in echelon form

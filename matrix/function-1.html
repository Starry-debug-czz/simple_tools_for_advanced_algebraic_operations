对相同维度的两个矩阵进行逐元素加法运算。支持任意大小的实矩阵，并提供逐步可视化。 Add two matrices of the same dimensions with element-wise operations. Supports real matrices of any size with step-by-step visualization.

核心实现
Core Implementation


Matrix MatrixAddition(const Matrix& A, const Matrix& B) {
    // Validate matrix dimensions
    if (A.rows != B.rows || A.cols != B.cols) {
        throw std::invalid_argument("Matrix dimensions must match");
    }
    
    // Create result matrix
    Matrix result(A.rows, A.cols);
    
    // Perform element-wise addition
    for (int i = 0; i < A.rows; i++) {
        for (int j = 0; j < A.cols; j++) {
            result.data[i][j] = A.data[i][j] + B.data[i][j];
        }
    }
    
    return result;
}

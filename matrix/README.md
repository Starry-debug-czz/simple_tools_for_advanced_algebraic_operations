## Getting Started

### Local Development
1. Clone or download the project files
2. Navigate to the project directory
3. Start a local server:
   ```bash
   python -m http.server 8000
   ```
4. Open `http://localhost:8000` in your browser

### Usage
1. **Homepage**: Click "Enter the Matrix" to access the function directory
2. **Directory**: Select from 9 available linear algebra functions
3. **Function Pages**: 
   - Enter matrices in the input areas
   - View code implementation on the left
   - See step-by-step calculation process
   - Download results and processes

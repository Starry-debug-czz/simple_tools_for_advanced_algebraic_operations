核心实现
Core Implementation


double determinant(const Matrix& mat) {
    const int n = mat.rows;
    
    // Base cases
    if (n == 1) return mat.data[0][0];
    if (n == 2) {
        return mat.data[0][0] * mat.data[1][1] - 
               mat.data[0][1] * mat.data[1][0];
    }
    
    // Recursive expansion by first row
    double det = 0;
    for (int j = 0; j < n; j++) {
        Matrix submat = getSubmatrix(mat, 0, j);
        double sign = (j % 2 == 0) ? 1 : -1;
        det += sign * mat.data[0][j] * determinant(submat);
    }
    
    return det;
}

算法步骤
Algorithm Steps

1. 基本情况：直接处理 1×1 和 2×2 矩阵
1. Base Cases: Handle 1×1 and 2×2 matrices directly
2. 递归展开：沿第一行使用余子式展开
2. Recursive Expansion: Expand along the first row using cofactor expansion
3. 余子式计算：对每个元素计算其余子矩阵的行列式
3. Minor Calculation: For each element, compute the determinant of its minor matrix
4. 余子式和：求和：元素乘以其余子式 (-1)^(i+j) × a_ij × M_ij
4. Cofactor Sum: Sum elements multiplied by their cofactors: (-1)^(i+j) × a_ij × M_ij
